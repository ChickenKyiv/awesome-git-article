#### I want to list all branches

List local branches

`$ git branch`
List remote branches

`$ git branch -r`
List all branches (both local and remote)

`$ git branch -a`

---

#### Create a branch from a commit

`$ git checkout -b <branch> <SHA1_OF_COMMIT>`
